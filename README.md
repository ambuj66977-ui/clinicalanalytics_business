# ClarityRX3 Website

Production-ready Next.js + Tailwind website for ClarityRX3.

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run typecheck
npm run build
```

## Deploy on Vercel (Hobby)

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Keep default framework and build settings.
4. Add env var `NEXT_PUBLIC_SITE_URL` with your production domain.
5. Deploy.
